# E-commerce Back End Starter Code

# Description
E-Commerce application is to build the back end for an e-commerce site. E-commerce was built using Express.Js API and configure it to use Sequelize to interact with a MYSQL database.

# Installation/Usage
1. Clone Repo
2. Install dependencies by running "npm i"
3. Log into mysql shell, run: `source db/schema.sql`, close sql, seed database by running: `npm run seed`, and start server: `npm start`
